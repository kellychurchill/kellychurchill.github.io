A client asked us to help them find a website content monitoring tool which would trigger an alert when content on a website has been modified.

A couple of options we found:

Versionista - http://versionista.com
Pros:
  • Web based
  • Highlighting
  • Notifications
  • Version tracking
Cons:
  • Their pricing chart only goes up to 2000 pages
  • 2000 URLs for $380/mo?! That's 18 cents a URL. 
++++++++++++++++++++++++++++++++++++++++++++++
FireFox Plugin - http://sourceforge.net/p/updatescanner/wiki/Home/
Pros: 
  • Free
  • Browser based notifications
Cons:
  •  We have to enter each URL manually
